Embed Code

To embed this player in your website, use the following HTML code:

<iframe 
    src="http://your-server:5000/player?video={{ video_id }}&embed=1" 
    width="640" 
    height="360" 
    frameborder="0" 
    allowfullscreen>
</iframe>

Responsive Embed

For a responsive embed that maintains aspect ratio:

<div style="position: relative; padding-bottom: 56.25%; height: 0; overflow: hidden;">
    <iframe 
        src="http://your-server:5000/player?video={{ video_id }}&embed=1"
        style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;"
        frameborder="0" 
        allowfullscreen>
    </iframe>
</div>
{% endblock %}
